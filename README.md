### 🚀 Proyectos Destacados

#### 🎬 [MovieNexus](https://github.com/David-Santiago-Gil/MovieNexus)
* **¿Qué es?** Aplicación web interactiva que funciona como un catálogo completo de películas consumiendo la API externa de **TMDB (The Movie Database)**.
* **¿Cómo funciona?** Permite realizar búsquedas en tiempo real, visualizar detalles y calificaciones de películas, reproducir avances de forma segura y gestionar una lista persistente de favoritos (almacenada en `localStorage`).
* **Tecnologías utilizadas:**
  `Angular` `TypeScript` `Vanilla CSS` `TMDB API` `Vercel`

#### 📝 [gestor_tareas](https://github.com/David-Santiago-Gil/gestor_tareas)
* **¿Qué es?** Plataforma Full Stack orientada a la administración estructurada y visual de tareas pendientes.
* **¿Cómo funciona?** Cuenta con un Frontend dinámico y un Backend que gestiona las operaciones CRUD para las tareas. Organiza elementos por estado (to-do, in progress, completed) y prioridad, facilitando la organización diaria.
* **Tecnologías utilizadas:**
  `Angular` `TypeScript` `Node.js` `Express` `MySQL` `Vitest` `Render/Railway`

#### ⚡ [Farm_Stack_Crud](https://github.com/David-Santiago-Gil/Farm_Stack_Crud)
* **¿Qué es?** Aplicación Full Stack que ejemplifica el uso del moderno y rápido **FARM Stack**.
* **¿Cómo funciona?** Cuenta con un Backend de alta velocidad y rendimiento construido en **FastAPI** (Python) que expone endpoints REST, consumidos asíncronamente por una interfaz interactiva de **React** en el Frontend.
* **Tecnologías utilizadas:**
  `FastAPI (Python)` `React` `JavaScript` `Uvicorn` `MySQL / TiDB Cloud`

#### 🎰 [casino (NeonRoyaleCasino)](https://github.com/David-Santiago-Gil/casino)
* **¿Qué es?** Interfaz interactiva de entretenimiento digital con una estética llamativa inspirada en luces de neón retro-futuristas.
* **¿Cómo funciona?** Enfocado principalmente en el diseño visual y la experiencia de usuario. Implementa estilos CSS avanzados (texto resplandeciente, transiciones suaves y animaciones) para recrear una interfaz atractiva y responsiva con componentes web reutilizables.
* **Tecnologías utilizadas:**
  `Angular` `TypeScript` `HTML5` `CSS3 (Glow & Neon animations)` `Vercel`
